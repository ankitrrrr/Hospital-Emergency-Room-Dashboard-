# 🏥 Hospital Emergency Room Dashboard (Power BI)

## 📌 Project Overview

The **Hospital Emergency Room Dashboard** is an end-to-end Power BI project designed to analyze emergency room operations and provide actionable insights into patient flow, wait times, satisfaction, and hospital performance.

This project demonstrates the complete BI workflow — from requirement gathering to dashboard development — enabling stakeholders to make data-driven decisions and improve operational efficiency.

---

## 🎯 Objective

* Monitor emergency room performance
* Reduce patient wait time
* Improve patient satisfaction
* Optimize resource allocation
* Support data-driven healthcare decisions

---

## 📊 Dashboards Included

### 1️⃣ Monthly View

Provides month-wise analysis of ER performance.

**Key visuals:**

* Patient Admission Status (Admitted vs Non-admitted)
* Patient Age Distribution (10-year groups)
* Department Referral Trends
* Timeliness (Patients seen within 30 minutes)
* Gender Analysis
* Racial Demographics
* Patient Volume by Day & Hour

---

### 2️⃣ Consolidated View

Gives a high-level summary for a selected date range.

**Key insights:**

* Aggregated KPIs
* Trend analysis
* Overall hospital performance metrics

---

### 3️⃣ Patient Details View

Allows detailed patient-level exploration.

**Fields included:**

* Patient ID
* Patient Full Name
* Gender
* Age
* Admission Date
* Patient Race
* Wait Time
* Department Referral
* Admission Status

---

## 📈 KPIs Tracked

* Total Number of Patients
* Average Wait Time
* Patient Satisfaction Score
* Number of Patients Referred
* Timeliness %
